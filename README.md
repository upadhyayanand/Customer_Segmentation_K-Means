# Customer Segmentation using K-Means

## Project Overview
This project performs customer segmentation using the K-Means clustering algorithm to group customers based on their purchasing behavior.

## Dataset
- Features used:
  - Annual Income (k$)
  - Spending Score (1â€“100)

## Approach
1. Data preprocessing and feature selection
2. Feature scaling using StandardScaler
3. Determining optimal clusters using Elbow Method
4. Applying K-Means clustering
5. Visualizing customer segments
6. Interpreting clusters for business decisions

## Results
- Customers were segmented into 4 meaningful groups.
- High-income, high-spending customers were identified as premium customers.
- Insights can be used for targeted marketing and loyalty programs.
